# semilinear-pm-image-patches

Schmidhuber, Eldracher, Foltin, *Semilinear predictability minimization produces well-known feature detectors*, Neural Computation 8(4):773–786.

## Problem

Real-world image patches with no significant preprocessing or labels. Hidden units trained by predictability minimization develop V1-style receptive fields.

## What it demonstrates

First time bars-and-stripes-style 'natural image patches' appear in Schmidhuber's experimental corpus. Spontaneously develops oriented Gabor edges and center-surround detectors.
