# chunker-very-deep-1200

Schmidhuber 1993 Habilitationsschrift, *Netzwerkarchitekturen, Zielfunktionen und Kettenregel*, TUM.

## Problem

Deeper-stack version of the chunker scaled to credit assignment over ~1200 virtual layers / time steps.

## What it demonstrates

Schmidhuber's much-cited 1993 'Very Deep Learning' demonstration — credit assignment over 1200-step lags via the hierarchical predictor stack.
