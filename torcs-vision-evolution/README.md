# torcs-vision-evolution

Koutník, Cuccu, Schmidhuber, Gomez, *Evolving large-scale neural networks for vision-based RL*, GECCO 2013.

## Problem

TORCS car-racing simulator with raw-pixel input. Networks with >1M weights evolved in compressed (DCT-coefficient) weight space.

## What it demonstrates

First end-to-end vision-based TORCS controller learned by evolution. Compressed-weight-space evolution as a foundational technique.
