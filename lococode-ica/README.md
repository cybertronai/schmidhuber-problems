# lococode-ica

Hochreiter & Schmidhuber, *Feature extraction through LOCOCODE*, Neural Computation 11:679–714.

## Problem

ICA-like benchmarks: synthetic mixtures of independent sources, including a noisy bars problem.

## What it demonstrates

Low-complexity coding via flat-minimum search produces sparse, statistically-independent codes resembling ICA but with lower description length.
